# 変更仕様書: hashing() の新設と Spool 確定メソッド改名の追従

- 作成日: 2026-06-13
- 起点: docs/design/design-concept.md(To Do)
- 背景: レガシー移行(RHBCRT_REFPATH)の `.pl` が `$DBOBJ->hashing()` で素の AoH を期待する。現状は RHBCRT 側の `DBOBJ_bridge` が後付けしているが、DBOBJ 本体の恒久メソッドへ取り込む。あわせて、現行 Spool で動かない `spool()` の改名追従漏れを直す。

## 1. 改修一覧

| # | 内容 | 種別 |
| --- | --- | --- |
| 1 | `hashing()` を新設(非推奨・レガシー対応専用。廃止予定ではない) | 新 API |
| 2 | `spool()` の `Spool::lines` → `Spool::line` 追従 | バグ修正 |

## 2. hashing() の新設(改修1)

仕様の正本は design-concept「データ取得」「API」「Rules」。実装観点:

- シグネチャ: `$dbo->hashing()` → ハッシュリファレンスのリスト(AoH)を返す。0件で空リスト。
- 実装: `return @{ $self->hashes()->toAoh() };`。`hashes()` の薄い変換であり、undef→`''`・カラム情報・0件の扱いはすべて `hashes()` に従う(独自の取得処理を持たない)。
- **位置づけ: 非推奨・レガシー対応専用。** metaAoh ではなく素の AoH を直接欲しいレガシー資産のために残す。新規コードは `hashes()` を使う。コメント・design-concept(Concept/API/Rules)・spec.md に非推奨である旨を明記する。`廃止予定` ではない(将来の廃止は別途検討)。
- レガシー専用のため効率最適化はしない(metaAoh を構築して `toAoh()` で平坦化する経路のまま)。

## 3. Spool 確定メソッド改名の追従(改修2)

- 現行 Spool の確定メソッドは `line`(単数)・`records`・`grouping`。DBOBJ の `spool()`(`src/DBOBJ.pm` 187行)は改名前の `Spool::lines` を呼んでおり、引数なし確定で `Undefined subroutine &Spool::lines` の die になる。
- 修正: 187行 `Spool::lines($spool_id)` → `Spool::line($spool_id)`。`records`・`grouping` は改名されておらず現状一致のため変更なし。
- design-concept 内の旧名 `lines`(確定モードの呼称)を `line` に統一済み(Concept・API・Rules)。

## 4. テスト方針(test/dbobj.t)

| 対象 | ケース |
| --- | --- |
| hashing 基本 | 2件の SELECT 後 `hashing()` が素の AoH を返す。各要素が `{id=>.., val=>..}` のハッシュリファレンス。要素数=行数。metaAoh ではない(`MetaAoh::is_metaAOH` が偽) |
| hashing 0件 | 0件 SELECT で `hashing()` が空リストを返す |
| hashing 値整合 | 同一 SELECT で `hashing()` と `hashes()->toAoh()` が一致する(薄い変換の確認) |
| spool line 確定(回帰) | 既存「spool lines 確定」サブテストが改名追従後に緑になる |

- 既存テストと同じく実 DB(`DBOBJ->new('develop')`)を用いる。

## 5. 下流(別プロジェクト・RHBCRT_REFPATH 側で実施)

- `hashing()` が本体に入るため `DBOBJ_bridge` は不要。`src/DBOBJ_bridge.pm` を削除し、本体 `src/RHBCRT_REFPATH.pl` の `use DBOBJ_bridge;` を元の `use DBOBJ;` へ戻す。DBOBJ を再 bind し、`test/three_layers.t`(DBOBJ::hashing のモック検証を含む)が緑であることを確認する。
- これにより RHBCRT 側のブリッジが1つ・`# MIGRATION:` 差し替えが1行減る。
